### What this project does

- Home page with links to 3 roles
- Client: see conferences + register form
- Employee: see conferences + registered people (fake data)
- Admin: create / edit / delete conferences + edit users

No login, no real database – everything uses mock data.

### How to run

1. Clone repo  

2. Install packages  
   `composer install`  
   `npm install`

3. Start server  
   `composer run dev`

Open: http://127.0.0.1:8000
